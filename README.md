# Personal-Finance-dashboard
Personal Finance Dashboard built with React.js to track expenses efficiently. Features include add/edit/delete, category &amp; month filters, localStorage persistence, and dynamic Pie Chart visualization using Recharts. Clean, responsive UI with real-time total calculation.

# 💰 Personal Finance Dashboard (React)

A modern Personal Finance Dashboard built using React.js to track, analyze, and manage daily expenses efficiently.

---

## 🚀 Features

- ✅ Add, Edit, Delete expenses  
- ✅ Categories: Food, Rent, Fun, Travel  
- ✅ 📊 Pie Chart visualization (Recharts)  
- ✅ 📅 Filter by category & month  
- ✅ 💾 Data persistence using localStorage  
- ✅ 🌙 Dark Mode toggle  
- ✅ 🔍 Search functionality  
- ✅ 📁 Export expenses as CSV  
- ✅ 💵 Summary cards with total insights  
- ✅ Responsive UI  

---

## 🛠 Tech Stack

- React.js (Hooks)
- JavaScript (ES6+)
- Recharts
- CSS

## ⚙️ Installation & Setup

```bash
 Clone repo

# Go to folder
cd finance-dashboard

# Install dependencies
npm install

# Install chart library

npm run dev
